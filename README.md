# Portfolio v2

Made using Vue 3, Vite, Tailwind CSS, and GSAP.

## Features

- Vue 3 Script Setup Syntax (with TypeScript) and Composition API
- Vite
- Tailwind CSS
- GSAP
- TypeScript
- ESLint
- Prettier

## Usage

- Clone the repo
- Run `npm install`
- Run `npm run dev` to start the dev server
- Run `npm run build` to build the project
