# portfolio-v2

Made using Vue 3, Vite, and Tailwind CSS.

## Features

- Vue 3 Script Setup Syntax (with TypeScript) and Composition API
- Vite
- Tailwind CSS
- TypeScript
- ESLint
- Prettier
- EditorConfig

## Usage

- Clone the repo
- Run `npm install`
- Run `npm run dev` to start the dev server
- Run `npm run build` to build the project
